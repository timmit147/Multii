Hoelang duurt het maken van een website?

Het maken van een website kan al binnen 2 weken klaar zijn, afhankelijk van de grootte van de website.

Kan ik zelf de website beheren nadat deze is gebouwd?

Het aanpassen van de website gaat minder makkelijk dan bijvoorbeeld een WordPress of Wix website. Wanneer je een aanpassing wil aan de website, kan je dit voor een goedkope prijs door ons laten doen. Wanneer je echt zelf de website wil aanpassen, kunnen we daar een makkelijke uitleg voor geven zodat je zonder hulp kan doen.

Is de website te gebruiken op mobiel?

Ja, de website kan worden gebruikt op mobiele telefoons, tablets en computers, en op alle formaten daartussen.

Is de website te gebruiken op alle browsers?

Ja, de website kan worden gebruikt op verschillende browsers zoals Google Chrome, Safari, Firefox of Edge.
